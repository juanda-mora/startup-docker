# Startup Docker

Este repositorio contiene la dockerización de un proyecto web desarrollado como parte de la asignatura de Sistemas Operativos.

El proyecto utiliza Java para la lógica principal de la aplicación, junto con HTML y CSS para la interfaz web. Además, incluye archivos de configuración como `Dockerfile` y `docker-compose.yml`, que permiten construir y ejecutar la aplicación dentro de contenedores de Docker de forma más organizada y portable.

## Tecnologías usadas

- Java
- HTML
- CSS
- Docker
- Docker Compose
- Maven

## Objetivo del proyecto

El objetivo principal del proyecto es practicar el proceso de dockerización de una aplicación, entendiendo cómo empaquetar un sistema con sus dependencias y ejecutarlo en un entorno controlado mediante contenedores.
